10 Сентября 2007: Enigma 1.01 для 64-разрядных версий Linux

К несчастью Enigma содержит версию библиотеки zipios++, которая не совместима с 64 разрядами. Ни Windows, ни Mac OSX, ни Debian-основанные сборки не подвержены этой проблеме. Для всех остальных систем Unix, которые собраны непосредственно из выпущенных нами исходных кодов, мы предлагаем обновленный пакет исходных кодов, который уживается с 64-разрядными процессорами так же хорошо как и с 32-разрядными. Пожалуйста соберите выпуск с исправлениями "совместимости исходного кода с 32/64 разрядами" из нашего раздела загрузок.
