Sep 8 2007: Enigma 1.01 für 64-Bit Linux-Versionen

Einige Nutzer können Probleme beim Start von Enigma bekommen, wenn sie auf einem 64-Bit-System arbeiten. Wenn Sie diese Fehlermeldung sehen: "Cannot load application score file [...] Invalid document structure", kompilieren Sie den fehlerbereinigten Release "Quellcode 64-Bit" von unserer Downloadseite.
