Sep 9 2007: Enigma 1.01 für 64-Bit Linux-Versionen

Leider hat sich die in Enigma enthaltene zipios++ Library Version als inkompatibel zu 64-Bit herausgestellt. Weder Windows noch Mac OSX noch Debian basierte Compilationen sind von diesem Problem betroffen. Alle anderen Unix Systeme die direkt unsere Source Releases compilieren bieten wir eine überarbeitetes Source Packet an, dass sowohl 32-Bit als auch 64-Bit Prozessoren gerecht wird. Bitte compilieren Sie das fehlerbereinigte Release "Quellcode 32/64-Bit" von unserer Downloadseite.
