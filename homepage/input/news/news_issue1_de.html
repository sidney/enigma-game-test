Feb 16 2003: Enigma 0.70 ist released!

Download von unserer Download-Seite. Hier eine kurze Liste von Änderungen in diesem Release:

  • Geschwindigkeitsverbesserungen
  • Stereo-Sound
  • Mehr als 70 neue Levels
  • Stark verbesserte Maussteuerung
  • Support der originalen Soundfiles von Oxyd (lesen Sie für Details die README Datei, die Enigma beiliegt)
  • Neue Spielobjekte wie Gummibänder, Wurmlöcher, bewegliche Plattformen
