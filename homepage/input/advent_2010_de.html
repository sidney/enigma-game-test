Advent 2010

Enigma feiert Advent!

Auch wenn es manchem so erscheinen mag, als wäre es um Enigma ruhig geworden – schließlich gab es seit über einem Jahr keinen LoTM, ebenso fand letztes Jahr keine Jahresendauszeichnungen statt, – täuscht dieser Eindruck.

Wir präsentieren dieses Jahr den ersten Enigma-Adventskalender mit 24 neuen – bisher unveröffentlichten – Level. Und wie es sich für einen richtigen Adventskalender gehöhrt, wird pünktlich am 1. Dezember das erste Türchen, hinter dem sich unser erstes Level verbirgt, geöffnet. Tag für Tag wartet ein weiteres Level darauf, nicht länger hinter einer verschlossenen Türe sein Dasein fristen zu müssen.

Participation works as follows. You need to download the levelpack index once from this URL "...". Copy the file to the given location, which is dependent on the operating system you run. For Windows it's "PATH_MS", for Mac OS X it's "PATH_OSX" and finally for Linux it's "PATH_UNIX". Then just start Enigma and enjoy the levels, each day you'll find one more in the "Advent 2010" levelpack.


Wir wünschen Euch viel Spaß mit unserem Enigma Adventskalender.
