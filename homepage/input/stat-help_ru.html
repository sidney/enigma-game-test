Пояснения

всего

В подсчёте участвуют все распространяемые уровни. Если для уровня доступен упрощённый режим, учитываются обе версии. Уровни, которые появляются несколько раз в разных пакетах уровней считаются за один.

совместных

Результат измеряется с разрешением в 1 секунду. Если двое или больше пользователей завершили уровень с точностью до секунды у них у обоих будет одинаковый результат и они разделят между собой мировой рекорд. Рекорд засчитывается как мировой обоим конкурентам.

гандикап (hcp)

Для игроков, которые не заостряют своё внимание на мировых рекордах, но всё же стремятся найти прямое и быстрое решение уровней, мы ввели другую меру. Мы назвали ее гандикап из-за схожести с гандикапом в гольфе. В общем меньшее значение лучше, чем большее. Диапазон колеблется от +100 до -300. Игрок, который решает уровни за среднее время получает гандикап 0.0. Гандикап представляет сумму результатов решений игроков из средних профессиональных результатов (PAR), которые имеются для каждого уровня. Чтобы сделать значения гандикапа сопоставимыми они всегда нормализуются к 100 уровням — стандартному размеру пакета уровней.

гандикап для решённых уровней

Гандикап для решённых уровней это гандикап, который рассчитывается для всех уровней, которые решил игрок. Дополнительно, в случае неудавшейся попытки решения, в расчёт берутся 10% нерешённых уровней. Таким образом, гандикап для решённых уровней может использоваться для сравнения игроков с весьма отличающимися результатами решения. Он показывает насколько быстро игрок решает уровень.
