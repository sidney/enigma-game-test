Entwicklung

Tätig werden

Enigma wird von einer kleinen Gruppe Freiwilliger entwickelt. Wir arbeiten in unserer Freizeit, die naturgemäß recht begrenzt ist. Falls Sie uns helfen möchten, sind Sie jederzeit willkommen! Falls Sie Interesse haben, an diesem Projekt tätig zu werden, melden Sie sich bitte bei der Entwickler-Mailingliste an und schreiben Sie uns.

Wie man die Entwicklerversion herunterlädt

Enigmas Subversion Repository wird momentan von Berlios gehostet. Für anonymen Zugriff, starten Sie einfach 

svn checkout svn://svn.berlios.de/enigma-game/trunk
Bitte kontaktieren Sie uns, wenn Sie Schreibzugriff auf das Repository benötigen.
