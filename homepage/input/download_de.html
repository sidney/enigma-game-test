Stabiles Release (Version 1.00)

Bitte berichten Sie alle Probleme mit einem dieser Pakete an die Enigma Mailing-Liste.

  • Windows '95 und später Vollständige Installation (1.00) etwa 15 MB
  • Windows '95 und später Minimale Installation (1.00) etwa 11 MB
  • Windows '95 und später Minimale Installation als Zip (1.00) etwa 11 MB
  • Mac OS X universell Volle Installation (1.00) etwa 16 MB
  • Quellcode tar.gz (1.00) etwa 10 MB
  • Levelvorschau-Bilder Zip (1.00) etwa 4 MB

Die Minimalinstallationen und der Quellcode enthalten nicht die Levelvorschau-Bilder. Allerdings werden fehlende Vorschau-Bilder direkt in Enigma erzeugt, was lediglich ein paar Sekunden bei den ersten Aufrufen des Levelmenüs kostet.

Alle Pakete stehen auch auf Berlios zur Verfügung.

Pakete für andere Betriebssysteme wurden von Dritten zur Verfügung gestellt:

  • Debian
  • FreeBSD (Version 0.92)
  • Gentoo Linux
  • Ubuntu
