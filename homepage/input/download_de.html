Stabile Version (Version 1.01)

Neben den 'vollständigen' Paketen stellen wir auch kleinere 'minimale' Pakete für User mit geringerer Bandbreite (z.B. Modem) zur Verfügung. 'Minimale' and 'Quelltext-Pakete' enthalten nicht das 'Vorschaubilder'-Paket. Sie sind dennoch vollständig, da fehlende Vorschaubilder direkt in Enigma erzeugt werden, was lediglich ein paar Sekunden bei den ersten Aufrufen des Levelmenüs kostet.

Die md5- und sha256-Zahlen sind Checksummen, mit denen Sie die Authentizität und Exaktheit Ihres Downloads überprüfen können, unter Verwendung entsprechender Programme. Für die Installation oder Nutzung von Enigma sind sie nicht von Bedeutung.

Quellcode kompilieren unter Unix

Falls es für Ihre Unix-Distribution keine Enigma-Pakete gibt, möchten Sie vielleicht die Quellpakete herunterladen und Enigma selbst kompilieren. Dies ist normalerweise ziemlich einfach.

Enigma hängt von einigen Bibliotheken und Werkzeugen ab. Die meisten Unix-Systeme enthalten heutzutage diese Bibliotheken bereits von Anfang an. Aber fürs Kompilieren benötigen Sie zusätzlich '-devel'-Pakete neben den üblichen Laufzeit-Paketen. Bitte stellen Sie sicher, dass die Bibliotheken SDL, SDL_ttf, SDL_image, SDL_mixer und Xerces auf Ihrem System installiert sind.

Für eine vollständige Liste aller Bibliotheken und Programme, die fürs Kompilieren benötigt werden, lesen Sie bitte die Datei 'doc/REQUIREMENTS' im Enigma-Quelltext-Paket.

Sobald Sie alles installiert haben, geschieht das Kompilieren einfach durch

./configure && make && make install

im Enigma-Verzeichnis.

Falls die benötigten Pakete nicht in Ihrer Distribution enthalten sind, können Sie sie auch auf den folgenden Seiten herunterladen:

  • SDL  (≥ 1.2)
  • SDL_ttf  (≥ 2.0.6)
  • SDL_mixer   (≥ 1.2.4)
  • SDL_image   (≥ 1.2.0)
  • Xerces  (≥ 2.4)

Ankündigungs-Mailingliste

Damit Sie stets auf dem neuesten Stand sein können, wenn eine neue Version von Enigma erscheint, bieten wir einen Benachrichtigungsservice via Email an. Tragen Sie sich dafür auf unserer Ankündigungs-Mailingliste ein. Das Mailvolumen ist beschränkt auf eine Mail pro Beta oder stabiler Version, was bislang nicht mehr als ein halbes Dutzend Mails im Jahr ausmachte.
