Получить Enigma

Все ссылки и некоторые советы по загрузке Enigma представлены на этой странице и отсортированы по следующим критериям:
  • Стабильный выпуск - Самые новые выпущенные пакеты - начните отсюда.
  • Сборка из исходных кодов - Советы для Unix.
  • Список рассылки с уведомлением о выпуске - Получать по почте уведомление о выходе нового выпуска Enigma.
  • Разрабатываемые версии - Скачать самые последние исходные коды.
  • Старые версии

Стабильный выпуск (Версия 1.01)

Сборка из исходных кодов под Unix

Если Вы не нашли пакетов Enigma для своего дистрибутива Unix, Вы можете попытаться скачать пакет с исходными кодами и собрать Enigma самостоятельно. Обычно это довольно просто.

Enigma зависит от нескольких библиотек и инструментов. Большинство Unix-систем поставляются с уже установленными необходимыми библиотеками. Но для сборки, помимо пакетов с выполняемыми библиотеками, Вам нужны дополнительные пакеты '-devel'. Пожалуйста проверьте, что в Вашей системе установлены библиотеки SDL, SDL_ttf, SDL_image, SDL_mixer и Xerces.

Чтобы увидеть полный список библиотек и программ, которые должны быть установлены в Вашей системе для сборки Enigma, пожалуйста, просмотрите файл 'doc/REQUIREMENTS', который включён в пакет исходных кодов Enigma.

Как только всё будет установлено, создание Enigma станет простым как набор команд

./configure && make && make install

в директории enigma.

Если в Вашем дистрибутиве нет необходимых пакетов, Вы можете скачать их со следующих веб-сайтов:

  • SDL  (≥ 1.2)
  • SDL_ttf  (≥ 2.0.6)
  • SDL_mixer  (≥ 1.2.4)
  • SDL_image  (≥ 1.2.0)
  • Xerces  (≥ 2.4)

Если Вы хотите собрать текущую разрабатываемую версию из SVN, Вам дополнительно понадобятся curl и libcurl4.

Если Вы используете Debian, то для сборки разрабатываемой версии из SVN у Вас должны быть установлены следующие пакеты:

  • curl
  • libcurl3-openssl-dev / libcurl4-openssl-dev
  • libsdl1.2-dev
  • libsdl-image1.2-dev
  • libsdl-mixer1.2-dev
  • libsdl-sound1.2-dev
  • libsdl-ttf2.0-dev
  • libxerces27-dev

Список рассылки с уведомлением о выпуске

Если Вы задались целью получить самую последнюю версию Enigma, мы предлагаем услугу уведомления о новых выпусках по электронной почте. Просто зарегистрируйтесь в списке рассылки с уведомлением о выпуске. Объем почтового трафика будет ограничен одним сообщением о бета или стабильном выпуске Enigma, который ещё не превосходил полудюжины сообщений в течение года.

Разрабатываемые версии

Заинтересованы в самой последней разрабатываемой версии Enigma? Разрабатываемую версию Enigma можно скачать с её репозитория исходных кодов.

svn://svn.code.sf.net/p/enigma-game/source

За подробными инструкциями прочтите, пожалуйста, страницу разработки.

Старые версии

Из-за изменений в репозитории исходных кодов и серверах загрузки, старые версии Enigma можно найти на различных серверах. Посмотрите на

  • sourceforge  если ищете недавние выпуски.
  • https://savannah.nongnu.org/download/enigma/  если ищете предыдущие выпуски
  • mag-heut.net  если ищете старые версии для Windows.
