Feb 16 2003: Enigma 0.70 ist released!

Download von unserer Download page. Hier eine kurze Liste von Änderungen in diesem Release:

  • Geschwindigkeitsverbesserungen
  • Stereo sound
  • Mehr als 70 neue Levels
  • Stark verbesserte Maussteuerung
  • Support der originalen Soundfiles von Oxyd (see the README file that comes with Enigma for details
  • Neue Spielobjekte wie Gummibänder, Wurmlöcher, bewegliche Platformen
